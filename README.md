## General info  
Calculator with styles/font/colors settings using JavaEE, JavaServer Faces, Java 8, CSS, Bootstrap 4.5.0, HTML5/XHTML, XML and server Glassfish 5.

* XML
* CSS
* Bootstrap 4.5.0
* JSF
* using server Glassfish 5.0
